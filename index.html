Inspetor de elementos

Com o recurso Inspecionar Elementos você pode visualizar o funcionamento de um site, ele faz parte das Ferramentas para o Desenvolvedor (Dev Tools) e nele é possível acessar vários recursos, como ver o código-fonte de uma página, alterar o HTML e CSS da página para testes1*, acessar um console (também para testes), rede, memória, etc.

Algumas formas de acessar o Inspecionar Elementos são:

  • Clicando com o botão direito do mouse e selecionando “Inspecionar” ou “Inspecionar Elemento”
  • Pela barra de ferramentas do navegador.
  • Por atalhos do teclado, Ctrl + Shift + C para o Windows, por exemplo.

1* A alteração do HTML e CSS serve para fazer testes, e são apenas momentâneas, apenas enquanto você está na página ou enquanto a página não é atualizada manualmente e apenas fica visível na sua máquina, não há alterações de fato na página original.

Sobre o HTML

A Linguagem de Marcação de hipertexto, ou seja, HTML, é a linguagem padrão para criar páginas da Web.

Ela descreve a estrutura de uma página e possui uma série de elementos, os elementos HTML por sua vez informam ao navegador como deve ser exibido o conteúdo da página.

Estrutura básica do HTML

Todos os documentos HTML devem começar com uma declaração de tipo de documento:

A declaração <!DOCTYPE> representa o tipo de documento e ajuda os navegadores a exibirem as páginas da Web corretamente.

O próprio documento HTML começa com <html> e termina com </html>.

Onde:

  • <!DOCTYPE html>: define que este documento é um documento HTML5
  • <html>: é o elemento raiz de uma página HTML
  • <head>: contém metainformações sobre a página HTML e onde fica o Título da página que é mostrada na barra de título do navegador
  • <title>: especifica um título para a página HTML
  • <body>: define o corpo do documento, onde fica todo o conteúdo visível, como títulos, parágrafos, imagens, links, tabelas, listas, etc.,
  • <h1>: define um título grande
  • <p>: define um parágrafo

Titulos:

Os Titulos HTML são definidos com as tags <h1> até a tag <h6>, sendo <h1> o titulo mais importante e o <h6> o menos importante.

Parágrafos

Os parágrafos são definidos com a tag <p>

Links

Os links são definidos com a tag <a>

O destino do link é especificado no atributo href.

Imagens

As imagens são definidas com a tag <img>

src é o atributo de origem da imagem, width e height são os atributos de largura e altura.

Listas

As listas podem ser usadas para listar itens ou até mesmo na criação de um menu, existem dois tipos de listas em HTML, a lista ordenada <ol> </ol> e a lista não ordenada <ul> </ul>. A diferença entre os dois tipos de listas é que, na lista ordenada, os itens são dispostos em uma lista numérica, e na lista não ordenada, os itens da lista são dispostos com um círculo preto/marcadores antes de cada item.
